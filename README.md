## Prerequisites

- GCC C++ compiler installed on your machine

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/cavallo5/cppscript.git
   ```
2. Compile the C++ source files:
   ```bash
   g++ -o main src/fileName.cpp -lcurl
   ```

## Usage
1. Start exec:
    ```bash
    ./main
    ```
